üöÄ Quick Installation

  1. Download and place as-banking in your resources folder
  2. Import database.sql into your MySQL database
  3. Add setr as_use_target "1" to server.cfg (or "0" for text UI)
  4. Add ensure as-banking to server.cfg
  5. Restart your server and enjoy!

üéÆ Interaction Modes

Target System as_use_target = "1"

  • Target ATMs and bank tellers
  • Clean interaction with icons
  • Eye/target icon appears on hover
  • Requires as-target resource

Text UI as_use_target = "0"

  • Distance-based interaction
  • Press E when prompted
  • ox_lib text UI notifications
  • Works without target system
